# Nick Arkhipov Portfolio

Minimal portfolio website for Nick Arkhipov.

Built with:
- Remix
- Three.js
- Framer Motion

## Local development

Requirements:
- Node.js 19.9+
- npm 9.6+

Install:

```bash
npm install
```

Run:

```bash
npm run dev
```

Build:

```bash
npm run build
```

## Deploy

Deploy to Vercel:

```bash
vercel --prod
```

## Notes

- Theme toggle is cookie-based.
- Project detail pages are ready for case study content.
